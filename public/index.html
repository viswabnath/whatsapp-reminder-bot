What Manvi can do

Everything you need,
nothing you don't

Seven core features designed around how you actually think and text — not how productivity apps want you to think.

⏰
Reminders
One-off or future-dated. Set a reminder in plain English — exact time, relative time, or a specific date. Manvi fires it at the right moment over WhatsApp.
"Remind me to check the oven in 20 minutes"
🔁
Daily Routines
Set a recurring task at a fixed time and it runs every day automatically — no expiry, no end date, until you delete it.
"Remind me to drink water every day at 10 AM"
🎂
Birthdays & Events
Save a date once, get reminded every year. Manvi sends an advance alert the day before and a fresh alert on the day itself at 8:30 AM.
"22nd May is Manu's birthday"
🔍
Web Search
Ask anything current. Manvi searches the web via Tavily (with Serper as fallback) and returns a clean, concise summary — no links, no fluff.
"Who won the match last night?"
📇
Contacts & Messaging
Save contacts by name. Ask Manvi to forward a message to anyone in your address book — she resolves the number and dispatches it instantly.
"Tell Ravi I'll be there at 7"
💬
Conversational AI
When you're not managing tasks, just talk. Manvi runs a 4-tier AI waterfall (Gemini → Groq → OpenRouter) and always keeps a fallback ready.
"Explain quantum computing simply"
How it works

From text to action
in under a second

1
You send a message
Plain English, in WhatsApp. No commands, no syntax. Just say what you need.
2
Manvi reads intent
The AI waterfall extracts intent, time, date, and target from your message in under 500ms.
3
Action is executed
The right handler fires — reminder saved, contact messaged, search run, routine scheduled.
4
You get notified
At exactly the right time, Manvi sends the reminder or event alert back to your WhatsApp.
Self-host in 15 minutes

Run your own instance.
Full control, zero cost.

Manvi runs on Render's free tier using free API quotas from Gemini, Groq, Tavily, and Meta. The only cost is your time to set it up.

Get your keys 5 min
1 Create a Meta developer app — get your PHONE_NUMBER_ID and ACCESS_TOKEN
2 Create a Supabase project — get your URL and anon key, run the schema SQL
3 Get free keys from Google AI Studio, Groq, Tavily, and Serper
→
Deploy 10 min
4 Fork the repo, deploy to Render as a Web Service, add all env vars
5 Set webhook URL in Meta dashboard to https://your-app.onrender.com/webhook
6 Add a cron-job.org ping every 10 min to keep the instance awake
→
Run locally optional
git clone <your-fork> cd manvi-whatsapp-assistant npm install cp .env.example .env npm run dev
